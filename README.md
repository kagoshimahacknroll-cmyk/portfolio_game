# 侍の領国経営 - Bushi Life

日本戦国時代の侍（大名）として領国を経営するブラウザシミュレーションゲームです。

## ゲーム概要

プレイヤーは戦国時代の大名となり、領国の発展と統治を行います。農業、軍事、政治、外交など様々な要素をバランス良く発展させ、強固な領国を築き上げることが目標です。

## 主な機能

### 🏛️ 領国管理
- **資源管理**: 米、金、人口、兵力、領地、名声の6つの主要資源
- **月次進行**: 毎月資源が増加し、ランダムイベントが発生
- **セーブ・ロード**: ブラウザのローカルストレージにゲームデータを保存

### ⚔️ 鍛錬・訓練
- **剣術訓練**: 個人の戦闘力を向上
- **兵法研究**: 戦略・戦術の知識を深める
- **体力鍛錬**: 持久力と健康を向上

### 🌾 農業・栽培
- **田畑開墾**: 新しい農地を開拓して米の生産量を増加
- **灌漑整備**: 水田の生産性を向上
- **新作物導入**: 新しい作物を栽培して収入を増加

### 🏛️ 政治・統治
- **税制改革**: 税収を安定させる
- **法制度整備**: 領内の治安を向上
- **外交活動**: 他国との関係を改善

### ⚔️ 軍事・防衛
- **兵士募集**: 兵力を増強
- **城郭建設**: 防御力を大幅に向上
- **武器製造**: 装備を改良

### 🎲 ランダムイベント
- **良いイベント**: 豊作、商人来訪、技術導入など
- **悪いイベント**: 疫病、盗賊、干ばつなど
- **有事対応**: 戦争、反乱、外交危機など

## ゲームの遊び方

1. **ブラウザで `index.html` を開く**
2. **大名名を設定**（デフォルト: 織田信長）
3. **各タブでアクションを選択**
   - 鍛錬: 個人能力の向上
   - 農業: 領国の生産力向上
   - 政治: 統治システムの改善
   - 軍事: 防衛力の強化
   - 事件: イベント履歴の確認
4. **「次の月へ」ボタンで時間を進める**
5. **資源のバランスを保ちながら領国を発展させる**

## 戦略のヒント

### 🎯 バランス重視
- 一つの資源だけに偏らず、全体的な発展を心がける
- 軍事力と経済力のバランスが重要

### ⚔️ 有事に備える
- 常に一定の軍事力を維持
- 城郭建設で防御力を向上
- 外交で他国との関係を良好に保つ

### 🌾 経済基盤の確立
- 農業投資で安定した食料供給を確保
- 税制改革で安定した収入源を構築
- 人口増加で労働力を確保

### 📈 長期的視点
- 短期的な利益より長期的な発展を重視
- イベントに一喜一憂せず、着実に発展
- セーブ機能を活用して安全にプレイ

## 技術仕様

- **HTML5**: ゲームの基本構造
- **CSS3**: 侍テーマの美しいUIデザイン
- **JavaScript (ES6+)**: ゲームロジックとインタラクション
- **ローカルストレージ**: セーブデータの保存
